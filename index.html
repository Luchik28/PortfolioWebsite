StudyCal

A web based Calendar app orientated towards students that organizes your day, and also critiques your schedule, with the goal of creating a more balanced, optimized schedule.

StudyCal has more than 200 Monthly Active Users as of January 2026, making it my most used project. It's been particurally gratifying to read all of the reviews and feedback from people who found it to be useful. One thing I learned from this project was about training AI models. Problem Scraper

December 2025

A Python Script that scans Reddit for potential project ideas. It searches for a problem, checks if it could be easily solved, and checks for existing solutions.

This project was inspired by needing to come up with new project ideas, and I wanted to create a tool to help automate this process. I first planned to build a web scraper, but this proved to be too complicated and frought with issues like finding a good API and also the results being filled with Ads. Instead, I just went with the Reddit API since it is free, reliable, and easy to use. I've used this API many times since then on other projects. I used an OpenAI API to analyze the problems for feasablity and Supabase for storage of ideas.

Robotics Telemetry Visualizer

November 2025 - February 2026

A tool I made for reading telemetry data from the robot during the 2025-26 Vex V5 robotics competition.

We won multiple Vex Think Awards, Vex Robotic's award for programming, for this feature. This is just part of the overall system that collected the data. First, the data was collected using sensors on the robot, including rotational sensors on tracking wheels, which recorded linear movement, and an IMU (Inertial Measurement Unit), which recorded rotational movement. That data is sent from the robot over the serial connection to the terminal, where it is picked up by a powershell script and written into a CSV file, that this application reads and displays. This system is useful because the robot uses this data for odemetry, which helps keep it on track when working autonomously. The robot is often innacurate, so knowing where the robot thought at was at all points during it's run made it a lot easier to debug.

AI Contribution Calculator

November 2025

A tool that scans your social media profiles to calculate how much data you're generating that can be used to train AI models.

I was inspired to create this tool after learning about how much value companies are getting out of the data users are generating on social media without the users realizing. For example, Reddit makes about 10% of it's revenue from data liscencing deals with AI companies. I decided to make this project in part to raise awareness of this phenomenon. I used a web scraper to find social media profiles related to someone's name, and then analyzed the content of those profiles to estimate the amount of data they were generating.
